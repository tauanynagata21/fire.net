# Fire.Net - Interface Inspirada em Streaming

Projeto desenvolvido como atividade avaliativa de conclusão de módulo, inspirado em plataformas como Netflix.

## Objetivo
Criar uma interface simulando um serviço de streaming utilizando HTML e CSS.

## Funcionalidades
- Layout com barra superior (header)
- Menu lateral com categorias
- Seção de recomendações e conteúdo
- Cards de filmes e séries clicáveis
- Estrutura visual inspirada em plataformas reais

## Tecnologias utilizadas
- HTML5
- CSS3 (Flexbox básico)

## Observação
Projeto desenvolvido como prática de layout e organização de interfaces web.
